개인정보 처리방침

최종 업데이트: 2026년 5월 11일

요약

SoDoIt(쏘두잇)은 사용자의 개인정보를 수집하지 않고, 외부로 전송하지 않으며, 제3자와 공유하지 않습니다. 모든 데이터는 사용자의 기기 내부에만 저장됩니다.

1. 수집하는 정보

SoDoIt은 어떠한 개인정보도 수집하지 않습니다. 사용자가 앱 안에서 입력한 할 일·메모·카테고리·마감일 등의 데이터는 모두 사용자의 기기 내부 (CoreData)에만 저장되며, 개발자나 제3자가 접근할 수 없습니다.

2. 외부 전송 및 공유

SoDoIt은 네트워크 통신을 수행하지 않습니다. 따라서 사용자 데이터가 외부 서버로 전송되거나 제3자와 공유되는 일이 없습니다. 인터넷 연결이 없어도 모든 기능이 정상 동작합니다.

3. 권한 사용

SoDoIt은 다음 시스템 권한을 사용합니다.

  • 알림(Notifications) — 사용자가 설정한 할 일 마감 시각에 로컬 알림을 전송합니다. 알림은 기기 내부에서 생성되며 외부 서버를 거치지 않습니다. iOS 설정에서 언제든 끌 수 있습니다.

4. 광고 및 분석 도구

SoDoIt은 광고를 표시하지 않으며, 사용 통계·크래시 리포트·행동 분석 등의 SDK를 포함하지 않습니다.

5. 위젯 데이터

홈 화면 위젯에 표시되는 할 일 정보는 App Group을 통해 동일 기기 내의 위젯 확장 프로그램에만 공유되며, 외부로 전송되지 않습니다.

6. 어린이의 개인정보

SoDoIt은 만 13세 미만 어린이를 대상으로 하지 않으며, 어떤 사용자의 개인정보도 별도로 수집하지 않으므로 어린이의 개인정보가 수집되는 일도 없습니다.

7. 정책 변경

본 처리방침이 변경될 경우 이 페이지에 변경된 내용과 시행일을 게시합니다. 중대한 변경이 있는 경우 앱 내 공지 또는 업데이트 노트를 통해 별도로 안내합니다.
